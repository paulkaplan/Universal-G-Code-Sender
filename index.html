This is a drop dead simple way to install and use the Universal Gcode Sender.

  • Download as a zip at the left.
  • Unzip and double click the .pkg installer.
  • Follow the instructions (admin password is required to make the /var/lock folder and change its permissions automagically for you)
  • Done!

Now in your /Applications folder you have the Universal Gcode Sender that you can launch like a normal app! Wooo! Hooray! Easy!
